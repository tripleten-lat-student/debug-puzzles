Puzzle Part 4. Stepping out

For this page, you'll need to use all the powers of the debugger, including the "Step out" button. Check behind each door to find the secret!

Tips & Tricks

  1. To open DevTools, you can right-click on the page and choose the "Inspect" option, or use the keyboard shortcut.

    • Mac: Cmd-Option-i
    • Windows/Linux: Ctrl-Shift-i

    Then refresh the page with the debugger open.

  2. If the debugger pauses without showing you the correct code, click the first entry in the Call Stack to jump to where it's really paused. This usually only happens the first time we pause on a given page.

  3. If it pauses on code that is obviously not part of the puzzle, this is probably due to chrome extensions. Just hit play, or open the page in an incognito window.
